Celeste Text Box Generator

Version 1.2

Quick and dirty Celeste text box generator made to create one single meme. It may be a bit slow since it is using the original game files in their original quality. Huge thanks to maddie480.ovh for hosting a graphics dump of celeste! This would not have been possible without it.

This generator has limited support for inline text commands (in the same format as the language files), including colored text, wavy text, shaky text, and the newline command. (You can still create new lines the regular way though!)

Supported Inline Text Commands

  • {~}Wavy Text!{/~}
  • {*}Shaky Text!{/*}
  • {# FFFF00}This text is bright yellow!{#} (uses HEX color values)
  • {+ CELESTE_MOUNTAIN} Inserts the text from the CELESTE_MOUNTAIN value (light blue text that says "Celeste Mountain")
    • Available values:
    • {n} (creates a newline without a line break)
